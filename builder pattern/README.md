# Builder Pattern

### Repo Link
[Repo](https://github.com/ShazaAllam2001/builder_pattern)

### User Class
contains fields:
- firstName (required)
- secondName (required)
- age (optional)
- phone (optional)

### User Builder Class
build user class in four stages:
- setFirstName
- setSecondName
- setAge
- setPhone
